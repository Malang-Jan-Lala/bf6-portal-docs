# Getting Started

This page gives a basic overview of the current BF6 Portal documentation.

## What this project covers

This project currently focuses on:

* Rules Editor concepts
* API/function reference data
* Selection Lists / Enums
* validated command behavior
* update workflow
* future searchable API/reference viewer

## What this project does not fully cover yet

Not complete yet:

* full Spatial/Godot map-building guide
* complete spawnpoint setup guide
* complete capture point / flag setup guide
* full vehicle spawn setup guide
* full end-to-end Portal experience tutorial

These topics are planned for later research.

## Data files

The `data/` folder contains cleaned reference data:

* API functions
* Rules Editor blocks
* Help data
* Selection Lists / Enum mappings

These files are useful for search tools, reference viewers, and future generated documentation.

## Important idea

The documentation separates three things:

```text
Extracted data
→ what exists in the API/editor data

Validated notes
→ what was tested in practice

Open research
→ what still needs testing or documentation
```

This helps avoid mixing facts, assumptions, and unfinished research.
