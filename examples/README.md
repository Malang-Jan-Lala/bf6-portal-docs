# Examples

This folder is reserved for future original examples and reusable patterns.

Examples are not the main focus yet.

## Goal

Future examples should help Portal creators understand common patterns such as:

* AI spawn control
* player kill missions
* capture point logic
* custom notifications
* music/audio events
* deploy control
* simple objective logic

## Rules

Examples should be:

* small
* original
* easy to understand
* focused on one pattern
* marked as tested or untested

Do not copy complete Creator scripts from BFPORTAL.GG, Discord, or other sources.

It is okay to learn from community examples, but public examples in this project should be written from scratch.

## Status

No public examples are included yet.

Examples will be added later after the documentation structure is clearer.
